# Debug Insight 🐞

A Django-based web app to track and manage debugging errors and solutions.

## Features
- Add and manage errors
- Tag-based search
- User authentication
- Clean UI

## Tech Stack
- Django
- HTML/CSS
- JavaScript

## Setup

'''bash
git clone https://github.com/yourusername/debug-insight.git
cd debug-insight
